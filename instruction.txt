1. Откройте командную строку (один из способов: ИСПОЛЬЗОВАТЬ КОМБИНАЦИЮ КЛАВИШ WIN + R, ЗАТЕМ ВВЕСТИ CMD И НАЖАТЬ ОК)
2. Укажите путь к корневому каталогу с проектом (cd путь к каталогу)
3. Активируйте виртуальную среду (conda activate practenv)
4. Установить зависимости с помощью команды pip install -r requirements.txt
5. Запустите программу с помощью команды python desktop_app.py
