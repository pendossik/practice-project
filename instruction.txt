1. Загрузите данный репозиторий и извлеките его 

2. Откройте командную строку:
    один из способов: ИСПОЛЬЗОВАТЬ КОМБИНАЦИЮ КЛАВИШ WIN + R, ЗАТЕМ ВВЕСТИ CMD И НАЖАТЬ ОК)

3. Укажите путь к корневому каталогу с проектом: cd <путь к каталогу>

4. Создайте виртуальную среду:
    1 способ: conda create -n practenv
    2 способ: python -m venv practenv

5. Активируйте виртуальную среду 
    для 1 способа в 3 пункте: conda activate practenv
    для 2 способа в 3 пункте: practenv\Scripts\activate

6. Установить зависимости с помощью команды pip install -r requirements.txt

7. Запустите программу с помощью команды python desktop_app.py
